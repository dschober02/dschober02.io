Airport Runway Simulation

This project involves coding an airport runway simulation that utilizes queues to manage and simulate the takeoff process for planes. The primary goal is to determine how long each plane waits for takeoff and to decide if a plane has enough fuel to fly as it awaits its turn. As planes are queued for takeoff, the simulation will decrement the fuel levels of each plane, and if a plane runs out of fuel before taking off, it will be flagged. The system will maintain a log of planes that successfully take off and those that are still in the queue, awaiting their turn. This simulation can provide valuable insights into the efficiency of runway operations and the impact of delays on aircraft fuel levels.

Social Media Platform Simulation

This program simulates a social media environment by implementing key design patterns. The Observer Pattern enables real-time updates between platforms and users, while the Strategy Pattern provides flexibility in search algorithms. A Factory Pattern dynamically creates instances of various social media platforms with unique characteristics. The program simulates realistic activity by pulling random feeds from a file and injecting them into each platform at its respective update rate. This approach enhances code reusability, maintainability, and adaptability, allowing for the creation of diverse and dynamic social media platforms within the simulation.
